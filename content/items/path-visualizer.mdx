The project began as a way to make graph traversal visible instead of abstract.

The useful challenge was not only implementing Dijkstra or A*; it was turning each algorithm step into a clear visual state. That meant thinking about grid representation, node state, animation timing, and how much feedback a user needs before the algorithm becomes understandable rather than noisy.
